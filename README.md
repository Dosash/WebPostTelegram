# WebPost

Простое веб-приложение для отправки постов в Telegram-каналы через ботов.
Поддерживает несколько ботов, списки каналов для каждого бота, HTML-текст,
картинки, кнопки, архив и логи.

## Возможности
- Несколько ботов с понятными именами
- Каналы привязаны к боту (отображаются каналы выбранного бота)
- Отправка текста (HTML) и/или картинки с кнопкой
- Архив с превью картинок и кнопкой “Edit in Send”
- Логи с подробным статусом
- Задержка между отправками
- Переключатель темы (по умолчанию темная)

## Требования
- Node.js 18+ (проверено на новых версиях)

## Установка
```bash
npm install
```

## Запуск
```bash
npm start
```

Открой `http://localhost:3000`.

## Как пользоваться
1) Добавь бота (имя + токен).
2) Добавь каналы для этого бота.
3) В разделе Send Post выбери каналы, бота и контент.
4) Проверяй Archive и Logs.

## Примечания
- Картинки сохраняются локально в `data/uploads`.
- Логи хранятся в `data/logs.json`.
- Архив хранится в `data/archive.json`.
- Конфиг ботов и каналов — в `data/config.json`.

## Структура проекта
